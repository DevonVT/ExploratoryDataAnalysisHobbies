Hobbies Dataset Exploratory Data Analysis (EDA) – Python Data Analytics Project
Technologies: Python, Pandas, MatPlotlib, Seaborn, Data Visualization
Analyzed and cleaned a lifestyle survey dataset containing 8,400+ records and 23 variables using Python and Pandas.
Performed exploratory data analysis to identify relationships between hobbies, demographics, and television viewing habits.
Developed 10+ visualizations and statistical summaries to uncover behavioral trends and participation patterns.
Designed an interactive Tableau dashboard featuring hobby participation rates, demographic comparisons, and lifestyle insights.
Created filters and dynamic visualizations enabling exploration by age group, sex, and marital status.
